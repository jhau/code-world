# PetStore (reconstruction)

Reconstructed from knowledge of `polcham/mongoose-express-ts` (the boilerplate
that `manish87sharma/PetStore` clones), because GitHub blocks automated access
to file contents from this environment. Structure and behavior are faithful to
the documented boilerplate (middleware / models / routes / types layering,
JWT auth, Users + Profiles CRUD); exact line-level contents are approximate.

Consequence for the IR: every fact carries `provenance: inferred` rather than
`parsed`, with confidence noted, and source spans are approximate. This is the
"agent frontend" path of the world-compiler architecture, exercised for real.
